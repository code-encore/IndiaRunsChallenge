# AI-Powered Candidate Ranking System

## Problem Statement
Build a ranking system to identify the top candidates for a Senior AI Engineer role from 100,000 candidate profiles.

## Approach
- Profile Analysis
- Skill-Based Scoring
- Career History Analysis
- Recruiter Signal Analysis
- Candidate Ranking

## Features Used
- Years of Experience
- Current Role
- AI Skills
- Career History
- GitHub Activity
- Recruiter Interest
- Interview Completion Rate

## Output
Top ranked candidates exported as CSV.

## Tech Stack
- Python
- Pandas

- Dataset not included due to size limitations.
The project expects:

data/candidates.jsonl
